# codex-switcher-updates
Public update manifest for Codex Switcher

## 发布 update.json
1. 打开 `Actions`，运行 `Publish Update Manifest`。
2. `version` 填 `codex-switcher/package.json` 的 `version`（必须一致）。
3. 填写 `download_url`（蓝奏云分享链接）和 `notes`（可选）。
4. 生成文件：`update.json`。

原始地址：
- `https://raw.githubusercontent.com/wen495033653/codex-switcher-updates/main/update.json`
